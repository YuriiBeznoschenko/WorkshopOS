# WorkshopOS

Цифрова база майстерні.

## Головне правило

**Тільки підтверджені інструменти. Без припущень.**

Якщо модель не підтверджена — вона не потрапляє в інвентар.

## Правила ведення

1. GitHub є джерелом правди по майстерні.
2. Куплений інструмент додається в `inventory`.
3. Позиція з пошуку після покупки переноситься в `hunting/purchased.md`.
4. Не вигадувати модель, ціну, дату покупки, комплектацію або стан.
5. Якщо даних немає — писати `невідомо`.
