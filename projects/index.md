# Проєкти майстерні

Поточні проєкти, які треба вести окремими файлами.

## Активні / потенційні

- Мобільний стіл / верстак на колесах
- Фрезерний стіл
- Платформа / розширення для DeWALT D24000 під велику плитку
- Організація майстерні та зберігання

## Правило

Кожен проєкт має мати:

- ціль;
- «Нахуа?»;
- що вже є;
- що треба купити;
- креслення / розміри;
- статус виконання.
